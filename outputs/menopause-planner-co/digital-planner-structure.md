# Digital Menopause Wellness Planner - 19-Page Structure

## Digital Format Specifications

### Page Orientation
- **Landscape orientation** (16:9 aspect ratio)
- Optimized for tablet use (iPad, Android tablets)
- Compatible with GoodNotes, Notability, Noteshelf, PDF annotation apps

### Color Palette Reference
- **Sage Green**: #A5A58D (Navigation tabs, headers)
- **Dusty Rose**: #DDBEA9 (Accent buttons, highlights)
- **Soft Cream**: #FAF7F0 (Page backgrounds)
- **Warm Taupe**: #A39384 (Body text, subtle UI)
- **Deep Sage**: #6B705C (Primary text, icons)

### Hyperlink Navigation System
- Persistent navigation bar on all pages
- Tab-based navigation with visual indicators
- "Return to Home" button on every page
- Interactive checkboxes and dropdowns where applicable

## DIGITAL PRODUCTION NOTES

### Interactive Elements for PDF Apps
- Fillable text fields
- Clickable checkboxes
- Hyperlinked navigation buttons
- Dropdown menus (where supported)

### App Compatibility Notes
- **GoodNotes**: Full hyperlink support, handwriting input
- **Notability**: Hyperlinks work, audio note capability
- **Noteshelf**: Excellent for handwriting, hyperlinks supported
- **PDF Expert**: Full form fill capability
- **Xodo**: Cross-platform compatibility

### File Export Settings
- Format: PDF/A or PDF with embedded fonts
- Resolution: 150 DPI minimum for crisp display
- Color profile: sRGB for consistent screen display
- Hyperlinks: Embedded and tested before delivery

### Recommended File Size
- Target: Under 10MB for optimal app performance
- Compress images while maintaining quality
- Optimize PDF before distribution
