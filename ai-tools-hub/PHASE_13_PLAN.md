# Phase 13.1: AI Chatbot for Tool Recommendations

## Overview
Create a conversational chatbot widget that helps users discover AI tools based on their specific needs. Users ask questions like "What's the best AI tool for a content writer?" and the chatbot provides personalized recommendations with affiliate links.

## Architecture

```
User → ChatWidget (floating button) → /api/chat → Claude (AI Gateway) → tools.json
                                          ↓
                                    Parse response
                                    Find matching tools
                                    Return with affiliate URLs
```

## Technical Details

### Claude Prompt Strategy
```
You are a helpful AI tool recommendation assistant.
When users ask about their needs, recommend the 2-3 best tools from the database.
Be conversational but specific. Include:
1. Tool name and brief description
2. Why it's perfect for their use case
3. Pricing (if they ask)
4. 1-2 alternatives if relevant

Database context: [JSON list of 20 tools with categories, features, pricing]
```

### Tool Matching Algorithm
Option A: Let Claude identify tools naturally in response
Option B: Parse Claude response for tool names, then inject affiliate URLs

**Chosen**: Option A (simpler, more conversational)
- Claude naturally recommends tools by name
- We enhance the response with affiliate links using regex
- Pattern: Find tool names in response, append `[Get ChatGPT →](affiliate_url)`

### Streaming Implementation
```javascript
import { streamText } from 'ai';
import { toUIMessageStreamResponse } from 'ai';
import { createOpenAI } from '@ai-sdk/openai';

export async function POST(req) {
  const { message } = await req.json();

  // Route through AI Gateway with OIDC auth
  const client = createOpenAI({
    baseURL: process.env.AI_GATEWAY_URL,
    defaultHeaders: {
      'Authorization': `Bearer ${process.env.AI_GATEWAY_OIDC_TOKEN}`,
    },
  });

  const result = streamText({
    model: client('claude-3-5-sonnet-20241022'),
    system: SYSTEM_PROMPT,
    messages: [/* conversation history */],
  });

  return toUIMessageStreamResponse(result);
}
```

## Affiliate Link Injection
For each tool mentioned in Claude's response:
1. Find tool name in response (exact match or fuzzy)
2. Look up affiliate_url from tools.json
3. Replace tool mention with markdown link: `[Tool Name →](affiliate_url)`

Example:
```
Claude: "I'd recommend ChatGPT for..."
→ "I'd recommend [ChatGPT →](https://chat.openai.com?utm_source=toolpilot...) for..."
```

## Known Constraints
1. Claude context window: 200K tokens (plenty for tools list)
2. Streaming latency: ~500ms for first token
3. No persistent message history (resets on page reload)
4. Single-turn if no conversation state management
