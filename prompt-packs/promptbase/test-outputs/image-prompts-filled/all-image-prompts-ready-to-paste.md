## HOW TO USE THESE

1. Open Midjourney Discord (or Midjourney web app)
2. Paste each test prompt exactly as written into /imagine
3. Wait for the 4-image grid
4. Upscale the best one (U1-U4)
5. Save the upscaled image
6. Repeat for all 4 test variations per prompt
7. Pick the best 4 across all variations as your PromptBase cover images

For DALL-E 3 (Prompt 3):
1. Open ChatGPT with DALL-E enabled (or OpenAI API playground)
2. Paste each test prompt
3. Save the output
4. Pick the best 4 as cover images
